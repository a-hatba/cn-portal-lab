# cn-portal-lab

本仓库用于归档和发布多个独立的 HTML 页面。这些页面可视为不同主题、功能或实验性内容的静态展示，不针对任何特定域名或具体网站。

## 项目简介

cn-portal-lab 是一个轻量级的静态页面集合仓库。每个 HTML 页面自包含其样式与脚本，无需复杂后端支持，适合快速浏览、测试或作为示例参考。仓库本身不提供搜索、聚合或导航功能，仅作为页面的存储与分发点。

## 目录说明

- `pages/` —— 存放所有归档的 HTML 页面文件，每个文件独立命名。
- `assets/` —— 可选，存放页面引用的共用资源（如 CSS、JavaScript、图片等），但多数页面推荐内联资源以保持独立性。
- `docs/` —— 可选，存放与页面相关的说明或辅助文档。
- `README.md` —— 本文件，仓库的基本说明。

## 页面归档说明

- 所有页面均为静态 HTML，可直接在浏览器中打开运行。
- 页面内容可能包含实验性设计、功能演示或信息展示，但不代表任何正式服务或承诺。
- 页面文件名应尽量反映其内容或用途，避免使用无意义或误导性名称。
- 页面内不应包含外部跟踪、广告或恶意代码。

## 维护说明

- 欢迎通过 Issue 或 Pull Request 提交新的页面或改进建议。
- 添加新页面时，请确保其内容合法、合规，不侵犯第三方权益。
- 仓库维护者保留对页面内容进行审核、修改或删除的权利。
- 本仓库不保证所有页面长期可用或更新，请自行备份重要资料。

## 使用方式

直接通过 GitHub Pages 或本地文件系统访问 `pages/` 目录下的 HTML 文件即可。若需部署，可将整个仓库克隆至任意静态服务器。
